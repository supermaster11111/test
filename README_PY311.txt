CrunchyGod Project - Python 3.11 Environment
==========================================

1. Run `setup_py311.bat` to create the virtual environment and install dependencies.
2. Configure your secrets in `config/secrets.yaml` based on `config/secrets.example.yaml`.
3. Start the server with `run_server.bat` or `python run_server.py`.
4. Open http://127.0.0.1:5005/ in your browser to access the dashboard.

All screenshots and logs are stored under `data/outputs/screenshots` and `data/logs`.
